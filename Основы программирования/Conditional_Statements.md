<h1 align="center">Среди трех чисел найти среднее</h1>

> Вводятся три разных числа. Найти, какое из них является средним (больше одного, но меньше другого).
```diff
+ ЧислоА = 1;
+ ЧислоВ = 2;
+ ЧислоC = 2;
- CреднееЧисло = Неопрделено;
```
